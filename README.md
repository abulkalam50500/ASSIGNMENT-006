# DigiTools
Digitools is a simple user friendly tools marketplace where users can explore different digital products and manage their selections easily.

✨ Features
1. Add Product To Cart
2. Remove product form Cart List.
3. View All Cart List Product.
4. Transparent Pricing section with 3 plans:
  4.1 Starter
  4.2 Pro
  4.3 Enterprise

⚙️ Technologies Used
    * React
    * Tailwind CSS
    * DaisyUI
    * Vite
    * lusid icon
    * Material icon
    * Toastify

📌 Overview
This project ocuses on building a clean UI and smooth user experience for managing products and pricing plans.
